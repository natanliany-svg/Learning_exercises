מבט על: מושגי ליבה ב-JS 🚁

מה זה Closure?

פונקציה שיש לה "זיכרון". היא זוכרת את המשתנים מסביבת היצירה שלה גם אחרי שאותה סביבה סיימה לרוץ. זהו הכלי המרכזי ב-JS להסתרת מידע (Private State).

Factory Function

פונקציה שפשוט בונה ומחזירה אובייקט חדש בכל קריאה אליה. בניגוד ל-Classes, היא לא דורשת שימוש במילה new או this, מה שמקטין באגים.

Modules (ES6)

דרך לחלק את הקוד למספר קבצים. בעזרת export מוציאים פונקציות החוצה, ובעזרת import מכניסים אותן לקובץ אחר (בדומה ל-import בפייתון).

Closures & Factories 🔒

מה זה Closure?

קלוז'ר (Closure) קורה כשפונקציה פנימית "זוכרת" את הסביבה שבה היא נוצרה (Scope), גם אחרי שהפונקציה העוטפת סיימה לרוץ.

function createCounter() {
    let count = 0; // State is protected!
    return function() {
        count++;
        return count;
    }
}
const myCounter = createCounter();
console.log(myCounter()); // 1
0

Modules (ES6 vs CommonJS) 📦

CommonJS (Node.js ישן)

// ייצוא
module.exports = { myFunction };

// ייבוא
const { myFunction } = require('./file');

ES6 Modules (מודרני)

// ייצוא
export function myFunction() {}

// ייבוא
import { myFunction } from './file.js';
