הכרטיסיות והחומרים המקיפים לפיתוח Web מודרני בעברית.

🔗 פתח סימניות מקצועיות

📜 1. יסודות ומשתנים

// let: משתנה שאפשר לשנות (ניתן להשמה מחדש)
let count = 0;
count = 1;

// const: משתנה קבוע (לקריאה בלבד)
const user = { name: "Alice" };
user.name = "Bob"; // מותר! משנה את התוכן, לא את ההצבעה

// סוגי נתונים (Primitive Types)
const str = "Hello";    // מחרוזת טקסט
const num = 42;         // מספר
const isTrue = true;    // בוליאני (אמת/שקר)
const arr = [1, 2, 3];  // מערך (רשימה)

💡 טיפ: תמיד תשתמש ב-const כברירת מחדל. תעבור ל-let רק אם אתה יודע מראש שהערך של המשתנה ישתנה בעתיד (כמו מונה בלולאה).

🖥️ 2. שליטה במסך (DOM)

// בחירת אלמנטים מהמסך
const btn = document.querySelector('#myBtn');
const items = document.querySelectorAll('.item');

// מאזינים לאירועים (לחיצת כפתור)
btn.addEventListener('click', (e) => {
    e.preventDefault(); // מונע התנהגות ברירת מחדל
    console.log('הכפתור נלחץ!');
});

// יצירת אלמנטים חדשים והוספתם למסך
const div = document.createElement('div');
div.textContent = 'אלמנט חדש';
div.classList.add('highlight'); // הוספת מחלקת CSS
document.body.appendChild(div);

💡 טיפ: querySelector מוצא רק את התוצאה הראשונה, בעוד ש-querySelectorAll מביא לך רשימה של כל האלמנטים התואמים שתוכל לרוץ עליה עם לולאה.

✨ 3. כתיבה מודרנית (ES6+)

// פירוק מבנה (Destructuring)
const { name, age } = user; // שולף ישירות למשתנים
const [first, second] = arr;

// שילוב ופריסה (Spread Operator)
const newArr = [...arr, 4, 5]; // מוסיף למערך קיים
const newUser = { ...user, age: 30 }; // מעתיק אובייקט

// פונקציות חץ (Arrow Functions)
const add = (a, b) => a + b;

// מחרוזות חכמות (Template Literals)
const greeting = `שלום, ${name}!`;

💡 טיפ: שלוש הנקודות ... מאפשרות להעתיק או למזג מערכים ואובייקטים בקלות, חוסך המון שורות קוד לעומת פייתון.

🐍 VS 🟨 4. פייתון לעומת JavaScript

// פייתון: dict | ג'אווה-סקריפט: Object
// בפייתון מפתחות חיבים מירכאות, ב-JS לא
const dict = { key: "value" }; 

// פייתון: print() | JS: console.log()
console.log("מדפיס למסך");

// פייתון: def my_func(): | JS: function myFunc() {}
function sayHi() {
    // ב-JS חייבים סוגריים מסולסלים {} להגדרת בלוק קוד
    // ולא משתמשים בהזחות (אינדנטציה) כחובה
}

// בפייתון שוויון: == | ב-JS שוויון: ===
if (5 === 5) { /* תמיד תשתמש ב-3 פעמים שווה! */ }

💡 טיפ מעבר: ההבדל הכי קשה למעבר מפייתון הוא הסוגריים המסולסלים {} והנקודה-פסיק ; בסוף כל שורה. תתרגל לזה שסוף השורה לא נקבע על ידי שורת חדשה כמו בפייתון.

⏳ 5. אסינכרוניות (קוד שממתין)

// המתנה דמוית sleep בפייתון
const wait = (ms) => new Promise(resolve => 
    setTimeout(resolve, ms)
);

// פונקציה אסינכרונית
async function executeTask() {
    try {
        console.log('המשימה התחילה...');
        await wait(2000); // ממתין 2 שניות
        console.log('המשימה הסתיימה!');
    } catch (err) {
        console.error('שגיאה:', err);
    }
}

💡 טיפ: המילה השמורה await אומרת לתוכנית "תעצור כאן ותחכה שהפעולה תסתיים לפני שאתה ממשיך". ממש כמו בפייתון!

🌐 6. בקשות רשת ושרת (Fetch)

// מקביל לספריית Requests בפייתון
async function fetchUser() {
    try {
        const res = await fetch('https://api.example.com/user');
        
        // חייבים לבדוק ידנית אם השרת החזיר שגיאה
        if (!res.ok) {
            throw new Error(`HTTP Error: ${res.status}`);
        }
        
        // מקביל ל- res.json() בפייתון
        const data = await res.json();
        console.log('נתונים:', data);
    } catch (error) {
        console.error('הבקשה נכשלה:', error);
    }
}

💡 טיפ: פקודת ה-fetch לא זורקת שגיאה באופן אוטומטי אם השרת החזיר 404 (לא נמצא). חייבים לבדוק את התנאי res.ok כדי לוודא שהבקשה באמת הצליחה.
